# Команда /get_file_id

Скрытая утилита для администраторов бота, позволяющая получить `file_id` любого медиафайла (фото, видео, анимация, документ).

## Зачем это нужно?
`file_id` уникален для каждого бота. Если вы хотите, чтобы бот отправлял определенный файл (например, приветственную анимацию), вам нужно узнать его ID именно для **этого** бота.

## Как использовать?

1.  Отправьте боту в личные сообщения (или в чат) нужный файл (фото, видео, GIF).
2.  **Ответьте** (Reply) на это сообщение с файлом командой `/get_file_id`.
3.  Бот пришлет в ответ ID файла.

## Пример
1.  Вы отправляете `animation.mp4` как GIF.
2.  Делаете Reply на это сообщение: `/get_file_id`.
3.  Бот отвечает: `Animation ID: BQACAgIAAxkBA...`.

Этот ID можно использовать в переменных окружения (например, `LOGO_PIC`) или в коде.
