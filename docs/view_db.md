# Просмотр базы данных на сервере

Инструкция, как быстро посмотреть содержимое базы данных SQLite прямо на сервере через терминал.

## 1. Установка утилиты

Если команда `sqlite3` не найдена, установите её:
```bash
sudo apt update && sudo apt install sqlite3 -y
```

## 2. Настройка отображения (один раз)

Чтобы таблицы выводились красиво и с заголовками, выполните команду (создаст конфиг `~/.sqliterc`):
```bash
echo -e ".headers on\n.mode column" > ~/.sqliterc
```

## 3. Подключение к базе

Перейдите в папку проекта и откройте файл базы:
```bash
cd ~/iu8_queue_bot
sqlite3 db_data/queue_bot.db
```

## 4. Полезные SQL-запросы

Внутри консоли `sqlite>` вы можете выполнять запросы:

**Посмотреть список всех таблиц:**
```sql
.tables
```

**Посмотреть последних 10 активных пользователей:**
```sql
SELECT * FROM users ORDER BY last_seen DESC LIMIT 10;
```

**Посмотреть количество пользователей:**
```sql
SELECT COUNT(*) FROM users;
```

**Посмотреть список администраторов (кто создавал очереди):**
```sql
SELECT * FROM admin;
```

**Посмотреть список чатов:**
```sql
SELECT * FROM chat;
```

## 5. Выход

Чтобы выйти из консоли SQLite:
```sql
.quit
```
